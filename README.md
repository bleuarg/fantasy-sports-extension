# Fantasy Sport Extension
This extension adds a `Start active players` button on your Yahoo Fantasy team dashboard to start active players for every day of the season.

## Supported sports
- Baseball
- Basketball
- (American) Football
- Hockey

## Development
Development tested on node >= 6.

```
yarn
yarn start
```
or with npm
```
npm i
npm start
```
